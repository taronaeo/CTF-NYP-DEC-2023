# Games or No Games But Unlimited Games

**Challenge Category: Web** <br />
**Challenge Points: 500**

## Challenge Description

I recieved a secret message from the JONKLER in Arkham Asylum. It says: t-t-tHE wAy tHEE SeRvER renDeRs tEmpLAteS oN thE wEbSiTe iS wEiRD! Actually Jonkler Moment.

`games.nypinfosec.com`

## Analysis

### The Login

1. The website is statically generated with only a `Login / Register` button.
2. Any attempt to login using any combination of `username` and `password` would redirect you to `https://games.nypinfosec.com/home/<USERNAME>` and have the username reflected on the page itself.

### Plausible SQL Injection / XSS Attack

3. `<USERNAME>` data is always reflected back onto the webpage, possibly hinting either a SQL Injection or an XSS attack.
4. Using `ctf' OR 1=1--` result in the same values reflected back on the webpage, hinting that it might not be a SQL Injection attack.
5. Using `<h1>helloworld</h1>` result in a `404 Not Found` error, hinting that it might not be an XSS attack as well.
